1. 2차원 배열 순회
	- 행 우선 순회 : 2차원 배열의 행을 우선으로 순회
	- 열 우선 순회 : 2차원 배열의 열을 우선으로 순회

	- 2차원 행렬 전치 : 인접행렬을 전치하는 경우 유방향 그래프(정방행렬)에서 그래프의 방향을 모두 반대로 변경 
	* 행 우선 순회 : (row > column or row < column경우 변경)
	- 백준 : 2566번

2. 인접 행렬 : 모든 노드들의 다른 모든 노드들과 이어져 있는지 여부를 표현 
	- 간선이 노드보다 적은 경우 메모리를 많이 씀 
	- 가중치나 전치에 유용 

3. 인접 리스트 : 해당 노드와 이어진 노드들의 인덱스 번호를 직접 저장 (행렬보단 메모리 감소) / 배열 + ArrayList 
	- 가중치 정보 담기 어려움
	- 역방향 그래프를 만들기 어려움 

4. 완전 그래프 : 모든 간선이 연결 되어있는 그래프 

5. 델타 탐색 : 행렬을 순회하며 각 지점에서 상하좌우에 위치한 다른 지점을 조회하거나 이동하는 탐색 
	- 백준 : 1913번 문제 
	- 프로그래머스 : 신고 결과 받기
