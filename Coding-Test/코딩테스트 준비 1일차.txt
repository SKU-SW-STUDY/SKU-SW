=== 1차원 배열 알고리즘 
1. 구간합 [1, 2, 3, 4, 5]
	1) N개 구간 / (1개 구간당 M개) N X M => N^2 --> N, M의 개수가 많아지면 시간복잡도 비 효율  

	2) 슬라이딩 윈도우 알고리즘 O(N X M) -> O(N) 최적화 

	* 슬라이딩 윈도우 문제 적응 : 백준 2559번, 11659번 풀이 

2. String 알고리즘
	1) 투 포인터 기법 - 회문검사 : 백준 10988번
	2) Brute force 알고리즘 -> 무지성 검색 (완전탐색) : 백준 1541번 
	3) KMP 알고리즘 -> 코테에는 나오진 않음
